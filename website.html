Messenger Reforged

Målet med projektet var att skapa ett chattsystem som ska vara tilltalande till personer som sitter mycket med grupparbeten. Det innehåller de punkter som är viktigt med kommunikation i grupparbeten som egna grupper för kommunikation, dela filer med varandra, skapa event för att hålla koll på vilka personer som kan komma vid ett möte
